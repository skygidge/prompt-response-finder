### [9] "Type Setting" / "设计利器"
**ID:** `2026-06-03-ideogram-40-open-weight-design-model`  
**Category:** ai_art  
**Article date:** 2026-06-03  
**Source:** https://ideogram.ai/blog/ideogram-4.0/  

Ideogram released the first open-weight frontier text-to-image model on June 3 — a 9.3-billion-parameter Diffusion Transformer trained from scratch for design work, not general-purpose image generation.
• Weights on Hugging Face (nf4 and fp8 quantizations) and GitHub (ideogram-oss/ideogram4); inference code Apache 2.0; commercial deployment requires a paid license
• nf4 checkpoint fits on a single 24GB GPU — runs locally with no API cost
• Native 2K output with native alpha channels: transparent cutouts from inference, no background removal step
• JSON-based prompting: specify text placement, bounding-box layout, and color palettes (up to 16 hex values) — composition by specification
• #1 open-weight model on DesignArena; #9 overall, behind only OpenAI and Google closed models
• ComfyUI day-0 support
• Ideogram built its reputation on getting text right in AI images. It just put those weights on GitHub.
